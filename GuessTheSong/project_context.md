# Guess The Song

Platform:
Android

Goal:
Create a mobile music quiz game inspired by "Хто зверху?"

Core gameplay:
Player hears a short song fragment and guesses the correct answer.

Monetization:
- Ads
- Premium subscription

Requirements:
- Offline-first
- Minimal costs
- Fast MVP

Planned modes:
1. Classic Mode
2. Artist Mode
3. Lyrics Mode
4. Couples Mode
