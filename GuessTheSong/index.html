Create Custom Music Packs

Створення кастомних паків

Create custom packages with your favorite music! Follow the structure below or generate your metadata JSON automatically.

Створюйте унікальні набори пісень зі своєю музикою! Дотримуйтесь структури або згенеруйте файл метаданих JSON нижче.

How to Package:

Як створити пак:

  1. Prepare your song MP3 files (e.g. song1.mp3, song2.mp3).
  2. Підготуйте аудіофайли пісень у форматі MP3 (наприклад, song1.mp3, song2.mp3).
  3. Create a metadata.json file describing your tracks.
  4. Створіть файл metadata.json, який описує ці треки.
  5. Create an icon image named icon.png or icon.jpg.
  6. Додайте файл обкладинки пака з назвою icon.png або icon.jpg.
  7. Select all files and compress them into a ZIP archive.
  8. Виділіть усі файли разом та стисніть їх у **ZIP-архів**.
  9. Import the ZIP directly inside the app lobby!
  10. Імпортуйте ZIP-архів прямо в головному лобі гри!
